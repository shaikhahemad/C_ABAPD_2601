Practice artifacts and hands-on exercises built while preparing for the
**SAP Certified Associate – Back-End Developer (ABAP Cloud) (C_ABAPD_2601)** exam.

---

## 🎯 About This Repository

This repository contains ABAP Cloud development exercises covering all major
exam domains. All objects were developed using **ABAP Development Tools (ADT)**
in Eclipse on **SAP S/4HANA 2023 (ABAP 758)**, following ABAP Cloud–compliant
patterns and Tier-1 API usage only.

---

## 📂 Repository Structure

```
src/
├── oo_abap/         # OOP concepts — classes, interfaces, inheritance, exceptions
├── cds/             # CDS View Entities, associations, annotations, DCL
├── rap/             # RAP (unmanaged & managed) — BODef, handlers, validations
├── open_sql/        # Open SQL, HANA Code Pushdown, EML
└── odata/           # OData V4 service exposure via RAP
```

---

## 📘 Topics Covered

| Domain | Topics |
|---|---|
| **ABAP OOP** | Classes, Interfaces, Inheritance, Polymorphism, Exception Handling |
| **Core Data Services** | CDS View Entities, Associations, Annotations, Access Control (DCL) |
| **RAP** | Business Object Definition, Managed & Unmanaged, Determinations, Validations, Actions |
| **Open SQL & HANA** | Joins, Aggregations, Code Pushdown, EML |
| **OData V4** | Service definition, binding, Fiori Elements integration |
